---
description: Replicate the Republic/Santek merger simulation results
---

1. Open the terminal.
2. Navigate to the code directory: `cd code`
3. Run the specific simulation script: `Rscript TrashSims.R`

Note: This script requires `paired_data_4_22.csv` (check paths in script if it fails). It generates figures in `output/` (e.g., `TrashSimsFirm.png`) and tables in `doc/`.
